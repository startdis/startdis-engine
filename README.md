## 平台简介

Startdis Comm 即 Startdis Dev Tools 启迪快速开发平台所依赖的公共依赖模块。基于[Apache 2.0](https://www.apache.org/licenses/LICENSE-2.0)协议，毫无保留给个人及企业免费使用，由[江苏点九科技有限公司](https://dianjiu.cc)提供技术支持。
